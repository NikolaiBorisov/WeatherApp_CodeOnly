# WeatherApp_CodeOnly
WeatherApp stack: No Storyboards(code only), MVVM, RxSwift, UserDefaults.

Note: before you first run the App, please install the following pods: pod 'RxSwift', '6.2.0' , pod 'RxCocoa', '6.2.0'.
